TECHNOLOGIES

My language of choice is Python. I use it to solve complex problems and model data. Here are the rest of the tools and frameworks that I use.

Programming Languages

Compiled and scripted languages. For development, statistics, and predictive modeling.

Databases

Both SQL and NoSQL databases. To work with structured, document, key-value, and graph data.

Machine Learning

The core python packages. For cleaning and preprocessing data and creating predictive models.

Big Data

When Dealing with very large datasets. Hadoop and Spark are the standards of the industry to store and process data.

Other Tools

Other tools for automating deployments, versioning, visualizing data, and working in the cloud.
